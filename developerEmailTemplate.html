Project Update:
Sync Robot
 
Automate and a Home in the Cloud
 
 
The Sync Robot saves coaches hundreds of hours of double-entry reporting in district databases using Mule API’s, Selenium, and Web-GUI automation. Thanks to Matt (matthewjerome) for more than a year of development. Now we need help with:
 
  • Porting the service to a Cloud host
  • Designing Mule Automation and UI to run the sync
  • Implement Field Value-Mapping rules
  • Implement a Team-Mapping UI for Program Staff
